PTB land-atmosphere-ocean model (Clarkson 2015)

3-box Jorge L Sarmiento, J. R. Toggweiler (1984), J. R. Toggweiler, Jorge L Sarmiento (1985) ocean model, coupled to the COPSE land surface and sediment/crust.

Combination of forcings are as used in M. O. Clarkson, S. A. Kasemann, R. A. Wood, T. M. Lenton, Stuart J Daines, S. Richoz, F. Ohnemueller, A. Meixner, Simon W Poulton, E. T. Tipper (2015). See paper SI for model description.

NB: this is an untested PALEO configuration intended to reproduce the Matlab code from the original paper.

julia> include("PALEO_examples_PTB3box.jl")
